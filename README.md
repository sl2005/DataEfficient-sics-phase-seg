# Data-Efficient Surgical Phase Segmentation in Small-Incision Cataract Surgery

**A Controlled Study of Vision Foundation Models**

Lincoln Spencer, Song Wang, Chen Chen

Department of Computer Science, University of Central Florida

## 🌐 Project Website

**[https://sl2005.github.io/DataEfficient-sics-phase-seg/](https://sl2005.github.io/DataEfficient-sics-phase-seg/)**

## Overview

We study data-efficient phase segmentation for manual small-incision cataract surgery (SICS) through a controlled comparison of visual representations. We pair each visual encoder with the same temporal model (MS-TCN++) under identical training and evaluation settings on SICS-155 (19 phases), comparing supervised encoders (ResNet-50, I3D) against large self-supervised foundation models (DINOv3, V-JEPA2).

DINOv3 ViT-7B achieves the best overall results (**83.4% accuracy, 87.0 edit score**).
